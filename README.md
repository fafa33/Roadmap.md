# نقشه راه فنی ایراناواس (IranOS Roadmap) 🚀

هدف این نقشه راه، هدایت تلاش‌های تیم‌های برنامه‌نویسی برای پیاده‌سازی گام‌به‌گام سپیدنامه است. توسعه در ۴ فاز اصلی انجام می‌شود:

---

### فاز ۱: زیرساخت بلاکچین و لایه صفر (Zero-Layer)
*   [ ] **طراحی پروتکل اجماع:** انتخاب یا توسعه یک Sovereign Chain با امنیت بالا.
*   [ ] **پیاده‌سازی مکانیسم ماشه:** کدنویسی قراردادهای هوشمند (Smart Contracts) که به محض تشخیص تخلف، دسترسی‌های حاکمیتی را ابطال می‌کنند.
*   [ ] **توسعه لایه ZK-Rollups:** تست شبکه برای پشتیبانی از ۸۵ میلیون کاربر با کارمزد صفر.

### فاز ۲: اقتصاد دیجیتال و واحد پول پهلوی
*   [ ] **قرارداد هوشمند واحد پهلوی (Sp):** کدنویسی منطق انتشار پول با سقف ۹۰۰ میلیارد واحد.
*   [ ] **سیستم وثیقه‌گذاری (Vaults):** اتصال دیجیتال دارایی‌های صندوق ثروت ملی به پشتوانه پول.
*   [ ] **توسعه کیف‌پول ملی (National Wallet):** اپلیکیشن موبایل برای دریافت سهم رفاهی ۱۰۰۰ پهلوی توسط شهروندان.

### فاز ۳: اوراکل‌های ۳API و مدیریت هوشمند
*   [ ] **API 1 (پایش منابع):** ساخت رابط‌های نرم‌افزاری برای پایش شفاف استخراج نفت و معادن.
*   [ ] **API 2 (توزیع خودکار):** الگوریتم واریز مستقیم بودجه به استان‌ها و حقوق به شهروندان بدون واسطه دولت.
*   [ ] **API 3 (رتبه‌بندی تولید):** سیستم تحلیل داده برای تخصیص تسهیلات به کارخانه‌های بهره‌ور.

### فاز ۴: استقرار حاکمیت دیجیتال (Governance)
*   [ ] **سیستم رأی‌گیری ZK-Voting:** پیاده‌سازی انتخابات کاملاً شفاف و غیرقابل تقلب بر پایه هویت دیجیتال.
*   [ ] **دادگستری دیجیتال:** ایجاد درگاه یگانه برای ثبت احکام قضایی به صورت غیرقابل تغییر در بلاکچین.
*   [ ] **رونمایی از نسخه آلفا:** تست اجرای همزمان تمام فرگردهای منشور در محیط شبیه‌سازی شده.

---

## 📢 فراخوان داوطلبان
برنامه‌نویسانی که تمایل دارند در هر یک از این فازها پیش‌قدم شوند، لطفاً در بخش **Issues** آمادگی خود را اعلام کنند. 
**اولویت فعلی: پیاده‌سازی هسته مرکزی مکانیسم ماشه.**
